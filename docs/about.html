You turn your smart phone or computer on and open up Google Chrome. You can go to facebook to talk to your friends that are miles away, or you can go on Youtube and watch a video someone made years ago. How does that happen? How does the internet work? Who made the internet? This website will answer all of those questions.

First, you open up a web browser such as google and type in the URL that your are trying to reach.

The computer then looks for the websites IP using the DNS. It looks through caches, from the browser cache to the router cache and so on until it finds the address.

Third, the server and the browser communicate in private to protect your privacy.

The browser and server communicate by "speaking" in HTML.

Then the browser asks for the GET request, which is the HTML code for the web page.

The information is broken up into packets and sent to the browser.

Finally, the browser puts the packets together to form the requested web page.
